# MyOPC Frontend

MyOPC is a project-centered operating console for multi-agent work. It combines agent staffing, project knowledge, deployment monitoring, commerce workflows, integration health, and an intelligent coding workspace into one system.

## Run

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000` and enter the project center.

## Product Direction

- Project-first workspace: every page belongs to a project.
- Multi-agent management: agents are shared company resources assigned into projects.
- Knowledge cockpit: project notes plus workspace document search.
- Operations cockpit: domains, routing, monitoring, analytics, and release status.
- Coding system: MyOPC includes a local intelligent coding engine and execution entry.
